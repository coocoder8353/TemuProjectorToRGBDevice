[How To compile]
To install compilation tool you need to install python on your device
To Compile python code you need to paste this into your terminal to install the compilation tool: pip install pyinstaller
use cd to get to your file location for the python script
To comile use: pyinstaller --onefile TemuProjectorToRgbDevice.py
To run just launch the app.
